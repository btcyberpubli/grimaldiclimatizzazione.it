# Frio Renzo - Servizi di Condizionamento d'Aria

Sito web per Frio Renzo, azienda specializzata in:

- **Vendita** di impianti di condizionamento d'aria
- **Installazione** professionale di sistemi di climatizzazione
- **Manutenzione** e riparazione di impianti
- **Servizi linea bianca** - riparazione di elettrodomestici

## Contenuto del Sito

Il sito include le seguenti pagine in italiano:

- `indice.html` - Pagina principale con presentazione aziendale
- `chi-siamo.html` - Informazioni sull'azienda, missione e visione
- `servizi.html` - Panoramica dei servizi offerti
- `vendite.html` - Vendita di impianti di aria condizionata
- `installazione.html` - Servizi di installazione
- `manutenzione.html` - Servizi di manutenzione e riparazione
- `linea-bianca.html` - Riparazione di elettrodomestici
- `galleria.html` - Galleria di immagini dei lavori completati
- `contatti.html` - Pagina di contatto con moduli e informazioni

## Struttura Tecnica

- HTML5
- CSS3 (Bootstrap framework)
- jQuery per le funzionalità interattive
- Revolution Slider 5.4.7.1 per le presentazioni
- Formulario di contatto tramite JotForm
